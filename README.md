# Tarea-Programada-2

En este apartado explicaremos detalles generales sobre como compilar y ejecutar los comandos: 

Abrimos terminal

g++ *.cpp -o main
./main

veremos desplegadas las oraciones evaluadas con puntajes y se guardaran correctamente las palabras nuevas, de este modo cuando se ejecuten los mismos comandos nuevamente

g++ *.cpp -o main
./main

veremos cambios en el puntaje y evaluacion de las oraciones.

En el apartado del main, veremos los casos de prueba de las palabras nuevas y sus resultados esperados en contraste de los obtenidos, ahí también observaremos los casos fallidos y los posiblemente fallidos o acertados.
